# Feature Specs

Feature specs will live here after the agentic development framework is reviewed and accepted.

Do not start feature implementation until the workflow, prompts, review process, and branch policy are approved.

A future feature spec should usually include:

```txt
<feature>/
  current-behavior.md   # if legacy behavior exists
  requirements.md
  design.md
  interfaces.md
  test-cases.md
  tasks.md
```
